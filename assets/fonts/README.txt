Place custom font files (.ttf, .woff2, .otf) here.

To use a custom font in the editor:
1. Place your font file in this directory
2. Add an @font-face declaration in src/index.css
3. Add the font name to the FONT_OPTIONS array in src/components/InspectorPanel.jsx

The editor comes with several Google Fonts pre-loaded:
Bangers, Bebas Neue, Oswald, Anton, Russo One, Permanent Marker
